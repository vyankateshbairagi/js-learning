üíæ 1. localStorage

What is localStorage?

  • Stores data with NO expiration time
  • Data persists even after closing browser
  • Shared across all tabs/windows of same origin
  • Perfect for: User preferences, theme settings, app state

üîÑ 2. sessionStorage

What is sessionStorage?

  • Stores data for ONE session only
  • Data is cleared when page session ends (tab/browser closes)
  • Data is NOT shared between tabs/windows
  • Perfect for: Form data, temporary state, multi-step processes

üç™ 3. Cookies

What are Cookies?

  • Small text files stored on user's computer
  • Sent to server with every HTTP request
  • Can have expiration dates
  • Perfect for: Authentication tokens, tracking, server communication
